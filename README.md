## AFIT API

API para crear rutinas de gimnasio. (**En desarrollo**)

### Paquetes

- Django 5
- Django Rest Framework
- Django Rest Simple JWT

### Funcionalidades

- Autenticación JWT
- CRUD de usuarios
- CRUD de equipos
- CRUD de ejercicios
- CRUD de rutinas
- CRUD de entrenamientos

### Ejecutar app

```
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
